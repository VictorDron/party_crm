# Gala · CRM para casas de festa 🥂

Mock de um CRM completo para **casas de festa e salões de eventos**, em um único arquivo `index.html`
(HTML + CSS + JS, sem build). Do primeiro "oi" no WhatsApp ao último brinde.

> Produto irmão do **Giro** (CRM para revenda de veículos) — mesma suíte `agente.crm`.

## ✨ Destaques

- **3 salões, 1 painel** — visão consolidada do dono + visão individual de cada gerente (seletor de salão no topo, que escopa dashboard, funil, agenda, pagamentos e financeiro).
- **Funil de Reservas** — kanban arrastável com "travas de campo" que ensinam a equipe (um card não avança sem os dados obrigatórios).
- **Lead Inbox com IA** — todo lead que chega pelo WhatsApp Cloud é qualificado pela IA e vira card no funil. Você só aprova.
- **Pagamentos no Pix** — trilha de parcelas (mensalidade) + Agente de Cobrança que dispara o QR Code no vencimento e faz a régua de lembretes (D-3 / D0 / D+1 / D+3 / D+5).
- **Agenda dos salões** — calendário com detecção de conflito de data (cada salão faz uma festa por dia).
- **Pacotes & Preços** — tabela-mestre por tipo de festa (casamento, 15 anos, aniversário, corporativo, formatura) + planos SaaS.
- **Adicionais** — mesa de bolo, fotógrafo, painel de LED, banda… com selo *"fecha perto do evento"*.
- **Estoque & Equipe** — comida, bebida, garçom e músico com níveis mínimos e alertas.
- **Agente IA & Marketing** — campanhas e promoções via WhatsApp Cloud, reativação de clientes antigos (bodas, aniversários).
- **Relatórios** — faturamento por salão, leads por canal, funil de conversão, rankings.
- Tema claro/escuro, busca global (⌘K), gráficos (Chart.js) e design responsivo.

## ▶️ Como abrir

Basta abrir o `index.html` no navegador (duplo-clique ou `open index.html`).
Nenhuma instalação necessária — o único recurso externo é o Chart.js via CDN.
